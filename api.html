class linalgpy.core.solver.CoreSolver(evaluate=True)¶

核心求解器

参数

evaluate (bool) – 是否立即执行求解

dict() → dict¶

返回计算结果及过程，字典中的项目都是latex编码的字符串，类的子类可以调用此函数以返回一个字典

execute() → None¶

执行函数，该类的子类可以调用此函数以进行一次求解

toDict() → linalgpy.core.solver.CoreSolver.dict¶

todict()是dict的内层函数，不建议直接调用，一般通过dict()调用todict()
